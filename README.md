\# Fitness Tracker Application



A learning project built with:

\- Spring Boot (Java)

\- Angular

\- MySQL



Modules:

\- `fitness-api`: Backend (REST APIs, DB)

\- `fitness-ui`: Frontend (SPA)



Roadmap:

1\. Backend MVP

2\. Angular UI Integration

3\. JWT Security

4\. Charts, Goals, Maps (advanced)
